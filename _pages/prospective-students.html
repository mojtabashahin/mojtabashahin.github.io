The projects mainly target Software Engineering for AI (SE4AI) and AI for Software Engineering (AI4SE).

Project 1. Architecting AI-based Systems

This PhD project aims to support architecting AI-based systems by conducting empirical studies and developing practical solutions, design principles, and tools. More specifically, the project provides a deep understanding of architectural concerns and quality attributes in the development and deployment of AI-based systems and develops practical architectural solutions to design, evaluate, and monitor AI-based systems.

Project 2. DevSecOps-driven Software Architectures

This PhD research project aims to provide guidelines and tools with empirical evidence that allow software organisations to develop, apply, manage, and evaluate security architecture solutions and mechanisms in DevSecOps environments.

Project 3. Human Values and Human-centric Issues in Machine Learning Systems

The goal of this PhD project is to develop and evaluate well-established AI techniques to effectively detect, track, and classify human values and human-centric issues in machine learning systems by mining source codes and developer discussions captured in code comments and issue tracking systems.

Project 4. Socio-technical Aspects of Securing Machine Learning Systems

This PhD project aims to investigate security in machine learning systems from a socio-technical perspective. Specifically, it explores how and when developers should integrate and practice security in machine learning systems, how ML developers use security tools and security-related code recommended by AI tools, and what socio-technical factors impact AI developers while making security decisions.
